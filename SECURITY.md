# Security Policy

## Supported Versions

The current `main` branch receives security fixes.

## Reporting Issues

Please open a GitHub issue for security or safety concerns that do not expose private data. If an issue involves sensitive information, contact the maintainer privately instead of posting secrets or personal data publicly.

## Safety Scope

DeAI Humanizer should not be used to bypass AI detection, remove watermarks, strip metadata, hide provenance, or recover redacted private information. Reports about missing safeguards in these areas are welcome.
